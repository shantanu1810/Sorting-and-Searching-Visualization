Insertion sort is a comparison based sorting technique that build a final sorted array or list one item at a time.

It is efficient for small datasets and it is consider an adaptive sorting technique.

Time complexity of Insertion sort :
Best Case - O(N)
Average case - O(N2)
Worst case - O(N2)

Space complexity : O(1)
Where, N is the size of the array.
