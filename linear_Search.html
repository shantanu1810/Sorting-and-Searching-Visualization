In Linear Search Technique, target element are search by sequentially. It operates by sequentially checking all elements one by one starting from 0th index to the end index.

It is the most simplest searching technique, it just iterate over all the elements.

In this Visualization it will give you the first position and last position of the target element if it is present more then 1 time in the list.

Time Complexity of Linear search : O(N)
where, N = Size of the array.
