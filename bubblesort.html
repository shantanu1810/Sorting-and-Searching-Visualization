Bubble sort is a simple comparison based sorting algorithm. it repeatedly steps through the list, compares adjacents elements and swaps them if they are in the wrong order.

This process is continue until no swap is performed in a pass, means the the array is sorted.

Time complexity of Bubble sort : O(N2)
Space complexity : O(1)
Where, N is the size of the array.
