In selection sort, we first select the largest or smallest element in the array then put it in the first(0th index) or in the last index(according to ascending or descending order). Then we again find the smallest or largest element from the rest of the array(untill the array is sorted).

It is a comparison based sorting technique, it selects a smallest or largest element then swap it in its actual position.

Time complexity of Selection sort : O(N2)
Space complexity : O(1)
Where, N is the size of the array.
