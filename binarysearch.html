In Binary Search technique is a searching algorithm where the array are stored in Sorted order and we perform searching operation to find the target element.
Here we calculate the middle index formula : Mid = (low+high)/2
Here,
  • low = Array starting point(initially 0).
  • high = Array Ending point(initially size of array).
The low and high are changed according when binary search performs and if the target element is not present int the array then it returns -1 or "Not Found".

Time complexity of Binary search : O(log n).
